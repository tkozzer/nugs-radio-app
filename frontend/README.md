# Nugs Radio Frontend

This project is a Vue 3 application built with Vite, utilizing modern web development tools and practices.

## Tech Stack

- Vue 3
- Vite
- Tailwind CSS
- Pinia (for state management)
- VueUse

## Getting Started

### Prerequisites

- Node.js (version 14.0.0 or higher)
- npm (comes with Node.js)
